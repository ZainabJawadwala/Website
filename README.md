# 💎 AJ Jewels - Jewelry Website

- Overview
AJ Jewels is a responsive jewelry website designed to showcase various jewelry collections such as rings, chains, and buttons. The website provides a clean and modern UI for users to browse products and explore collections easily.

---

     Features
- Product listing with categories (rings, chains, buttons, etc.)
-  Product cards with "View Details" and "Add to Cart" buttons
- Responsive design for better user experience
- Contact section with phone, email, and social media
-  Clean and modern UI design

---
     Technologies Used
- HTML  
- CSS  
- JavaScript
